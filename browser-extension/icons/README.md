# 图标说明

请将插件图标文件放在此目录下：

- `icon-16.png` - 16x16 像素
- `icon-32.png` - 32x32 像素
- `icon-48.png` - 48x48 像素
- `icon-128.png` - 128x128 像素

## 设计建议

- 使用紫色渐变主题色 (#667eea 到 #764ba2)
- 图标应该简洁明了，易于识别
- 可以使用 Sparkit logo 或 "✨" emoji 风格
- 确保在浅色和深色背景下都清晰可见

## 快速生成图标

如果没有设计工具，可以使用在线工具：

1. **Canva** (https://canva.com)
   - 创建自定义尺寸画布
   - 使用渐变背景和简单图形
   - 导出为 PNG

2. **Figma** (https://figma.com)
   - 创建各种尺寸的 Frame
   - 设计图标
   - 导出为 PNG

3. **快速占位图标**
   - 可以暂时使用纯色 + 文字的简单图标
   - 后续再替换为设计版本

## 临时方案

在正式设计之前，可以使用 emoji 作为临时图标：
- 在任何图片编辑器中创建对应尺寸的画布
- 添加紫色渐变背景
- 放置"✨"或其他合适的 emoji
- 保存为 PNG
